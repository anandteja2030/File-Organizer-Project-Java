📁 File Organizer (Java)

A simple Java-based tool that automatically organizes files into folders based on their type (Images, Videos, Audio, Documents, etc.).


Features

* 📂 Automatically sorts files into folders
* 🖼️ Supports Images, Videos, Audio, Documents
* 📊 Displays total files moved
* 📁 Creates folders automatically if not present
* ⚡ Fast and lightweight


Technologies Used

* Java
* File Handling (`java.io.File`)


📂 Project Structure

File-Organizer-Java/
│
├── src/
│   ├── FileOrganizer.java
│   ├── Main.java
│
├── testfiles/
├── README.md
```

▶️ How to Run

1. Clone the repository

   git clone https://github.com/anandteja2030/File-Organizer-Java.git

2. Navigate to project folder

   cd File-Organizer-Java

3. Compile the program

   javac src/*.java

4. Run the program

   java src.Main
